What is it?

The Modular Grid provides a set of grid overlays in web pages so web authors can better implement translations of design comps to web pages. The user is able to create baseline, or leading, grids, column grids, and modular (simultaneous baseline and column) grids. Some key options include changing baseline distances, altering column and gutter widths, and updating baseline and column colors.

Install

The extension is available via the Chrome Web Store or Chrome’s Extensions page. New features are available in beta releases via GitHub.

Contributing

As an open source project, The Modular Grid is open to contributions from everyone, and you don’t need to be a web programmer. For example, we need German, French, Spanish, and Italian translations of The Modular Grid. If you are a web programmer, The Modular Grid uses HTML, CSS, Sass, core JavaScript, and Gulp. All open issues are filed on GitHub.

Feature Requests

Suggestions and feature requests, like bugs, may be submitted via the extension’s SUPPORT tab at Google’s Chrome Web Store or under the Issues tab on GitHub.

Bug Reports

There are two ways to file bugs: Under the extension’s SUPPORT tab at Google’s Chrome Web Store or under the Issues tab on GitHub. Bugs submitted at the Chrome Web Store will be copied over to GitHub.

For bugs related this web site, please visit GitHub.
